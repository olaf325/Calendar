# kalendarz
Kod wykorzystujący bibliotekę FullCalendar, żeby utworzyć kalendarz urlopów pracowników. 
Dane pobierane z pliku JSON formatowane są jako mapa, dzięki której tworzone są eventy dla każdego z nieobecnych pracowników.
Poprzez wprowadzenie do pola input daty i zatwierdzenia przyciskiem 'enter' ułatwione jest wyszukiwanie osób na urlopie.
